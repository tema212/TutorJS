Const


•Чтобы объявить переменную константой(неизменной) используется: Const


Const myAge = '18';


•Суть константы в том что ее нельзя изменить. Любое изменение приведет к ошибке

Const myAge = '18';

myAge = '20'; // Error


•Широко распространена практика использования констант в качестве псевдонимов для трудно запоминаемых значений, которые известны до начала исполнения скрипта. Названия таких констант пишутся с использованием заглавных букв и подчёркивания.

const COLOR_RED

const COLOR_BLUE

const COLOR_YELLOW


обычно данные константы используют для упрощения работы, так как их легче запомнить чем весь код
